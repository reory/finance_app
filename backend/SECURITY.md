# 🔐 Security Policy
I take the security of FinanceApp seriously.
If you discover a vulnerability, I'd appreciate your help in responsibly disclosing it.

---

# 📣 Reporting a Vulnerability
Please do not open a public GitHub issue.

---

Instead, report security concerns privately via:

# 📧 Email: reory35@hotmail.com

Include, if possible:

- A clear description of the issue

- Steps to reproduce

- Any relevant logs or screenshots

- Your suggested fix

I will review the report promptly and keep you updated.

---

# 🛡️ Supported Versions
Only the latest version of FinanceApp receives security updates.

---

# 🤝 Responsible Disclosure
Please give me reasonable time to investigate and fix the issue before any public disclosure.

I appreciate your help in keeping the project safe for all users
